Internet - куча компьютеров и серверов, объединенных вместе в одну сеть, которая еще называется web. ...

What is Server?
Сервера - хранят веб-сайты. И показывают их пользователям, когда те вводят в браузере веб-адресс, например http://facebook.com ...

• each server has web-address

Static web sites
• it's just a folder with web-page files
• each page of a site - it's different file
• web-page is written in HTML
• HTML - it's just a markup language
Эта страничка - чистый html, на ней есть заголовок, подзаголовок, параграф, картинка, ссылка, список. Каждый из этих элементов "обернут" в тег. Страничка выглядит не прикольно, потому что html используется для идентификации и показа контента, а не для стилизации и дизайна. Для того, чтоб страничка выглядела красиво - используется другой язык програмирования - CSS. Мы научимся использовать и HTML и CSS. Важно запонить что HTML - для содержимого, а CSS - для стилизирования ...
